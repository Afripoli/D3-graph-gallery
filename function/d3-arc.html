A tour of the d3.arc() function




The d3.arc() function is a shape generator, used to create arcs. It is part of the d3-shape module. Arcs can be created in pute svg, but this helper function will make the link between the data and the shape seamless.





ü¶ï Drawing an arc in pure svg

Drawing an arc is useful for a number of chart types, including the famous pie chart or its cousin the donut chart. It is totally doable to draw an arc in pure html, using an svg element.

Once you have set a svg element with a width and a height, call the path element inside it. A path is a powerful element allowing to draw any kind of shape. You can read more about it in the doc.

The shape of a path element is defined by one parameter: d. But as you can see in the example below, understanding the d language is not a piece of cake. It would be much easier to provide the coordinate of the arc center, its inner radius and its outer radius.

Good news! This is exactly what the d3.arc() function is made for!





üí° Most basic usage of d3.arc()

The d3.arc() function is here to create the content of the d parameter of the path element. When you call d3.arc(), it returns a function that we call a generator.

This generator is the function we dreamed of! It accepts an object with 4 properties and return the path we need. The arguments are innerRadius, outerRadius, startAngle and endAngle. Those names are self-explanatory üòÉ.
